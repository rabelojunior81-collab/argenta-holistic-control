**Key design decisions / DecisГөes de design:**
- **No frameworks** вҖ” plain Node.js HTTP + manual WebSocket. Zero hidden behavior.

- **Identity as context** вҖ” soul + skills injected as system prompt prefix. Flexible, not locked.
